Houdini

Day-to-day panel editing (add, rename, reorder, remove, switch) lives in the toolbar popup. This page holds the rest: behavior, backup & data, and the danger zone.

Behavior

Tab grouping

Uses Firefox's native tab groups. When you open a link in a new tab, it joins its opener's group (one level — a sub-tab adopts the opener's group). Flat list otherwise.

This Firefox version has no tab-groups API, so this option does nothing here.

Keyboard shortcuts

Click Set, then press your key combo (needs Ctrl or Alt). Esc cancels.

Backup & data

Automatic snapshots

Houdini periodically saves your panels and tab assignments. Roll back to any saved point.

Backup & restore

Save your panels and tab assignments to a file, or restore them on another machine. Restoring replaces all current panels and re-tags every open tab.

Import from Sidebery — replaces all current panels

Pick a Sidebery dump file, or paste the dump JSON below. Both replace your current panels and re-tag every open tab.

How to get a full Sidebery dump

about:debugging → This Firefox → Sidebery → Inspect → Console, paste this. It copies the JSON to your clipboard (paste it above) — a panels-only backup file works too but tags no tabs.
